# app voyage 3

> véritable compagnon de voyage, cette app permettra aux voyageurs de retrouver toutes les informations de son voyage ainsi que des éléments simplifiant la vie (traducteur, convertisseur,...). Option de partage de souvenirs de voyages avec ses proches. Proposée aux agences de voyages, travel planner ou meme directement b2c
